Share SRC

アバウト

  • このWEBアプリケーションはソースコードを共有するためのものです。
  • ソースコードにタイトルが付けられます。
  • ユーザー登録も必要なく、気軽に投稿できます。
  • また、共有したソースコードにはトークンが与えられ、それを/printに指定することで表示することができます。

特色

  • forkすることができます。
    • forkとは、他人のソースコードから新しくソースコードをつくることです。
  • このWEBアプリケーションにおいて共有されているソースコードを簡単にローカルにダウンロードできるPythonスクリプトを提供しています。
  • このPythonスクリプトのダウンロードはこちらからどうぞ。
  • オープンソースです。githubにてソースコードを公開しています。

github

  • このWEBアプリケーションのソースコードをgithubに公開しています
    • https://github.com/alice-himmel/ShareSRC

推奨環境

  • モニタ画面 - 1200*800(この画面よりも大きい場合は表示が小さいと感じるとおもいます)
  • 推奨OS - Linux, Mac OS X
  • 推奨ブラウザ - Firefox, Google Chrome (safariではコードの表示がすこしおかしくなります。)
